Кейсы и проекты

Практический курс "Создание 3D-игр в Roblox Studio"

Ситуация: В детском центре "Мозаика" требовался полностью практический курс по созданию игр для детей 10-14 лет без теоретических лекций.

Задача: Разработать 8-недельный курс, где на каждом занятии дети сразу создают элементы своей игры.

Действия: - Каждое занятие начиналось с демо-версии готового элемента;
- Пошаговое создание игры "Паркур с препятствиями" через практические задания;
- Использование Replit для написания скриптов Lua с готовыми шаблонами;
- Индивидуальные доработки проектов под интересы каждого ученика;
- Финальный хакатон с презентацией проектов родителям.

Результат: 100% учеников завершили курс с работающей игрой. Лучшие проекты были опубликованы в Roblox. Родители отметили высокую вовлеченность детей.

- Внедрение авто-доходов (функции, интервалы);
- Система улучшений (условия, события);
- Публикация проекта (GitHub Pages).

Результат: - 18 работающих игр с уникальным дизайном;
- 73% учеников продолжили изучать веб-разработку.
